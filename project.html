Formula 1: The Impact of R&D

In the 2021 Formula 1 season, the winning constructor (team), Mercedes, scored 613.5 points, which is 28% of the total points scored by all teams, whereas the bottom two teams scored 0.6% and 0% of the points respectively. Formula 1 is an immensely expensive sport and the amount of money that constructors have available to develop their cars likely has a significant impact on the performance of their cars during races. This project aims to evaluate the impact R&D expenditure has on this drastic difference in performance across teams.

Section 1: Yearly R&D Expenditure by Constructor

The data used here was accessed from a variety of sources (source list) and mainly includes motorsports blogs. The data was manually entered into an excel file. Given the variety of sources used and inconsistency in site design, a web scraper or API could not be used here. This posed the first challenge. The data used here are estimates as R&D data is not public. It is also not very replicable as new sites pop up every year with estimates for expenditure. However, given the frequency of data (1-year intervals) this is not a significant issue.


Section 2: Historical constructor standings

The data used here was accessed from the Ergast F1 API through Python using this script. Given the complexity of the API and multiple levels of nesting, it could not be directly accessed in Vega and Python had to be used. A module called pyErgast was designed for this purpose and heavily inspired the code used to obtain the data for this graph. The data in this section is automated and easily replicable as the same Python script can be run once a year to update the data when a season ends.


Section 3: Relationship between R&D and points scored

The data used here is the same as in Section 1 and 2. This presents the relationship between R&D expenditure and points scored. Each year, the number of races can vary, changing the maximum number of points available. To control for this, the variable used is the proportion of total points scored by each team. This was calculated in Python using this script. This shows a clear positive correlation.


Section 4: Relationship between stock prices and points

This presents a proxy for R&D expenditure, stock prices. The data was obtained using the yfinance Python module which accesses the yahoo finance API using this script. Only Mercedes and Williams have clear public parent companies and data going back to 2012, so they were the only teams selected. This was used as a proxy as the stock price of a team can be indicative of the value of the company and therefore the amount of funds available for investment in R&D. This shows that there is a clear correlation between average stock prices and race points for Mercedes in a year, but not for Williams.

Section 5: Qualifying position and race outcome

Finally, a small deviation. Given the complexity of predicting outcomes in Formula 1 races, this is a variable that I believe will strongly influence any models to predict race outcomes – Qualifying position. The day before a race each driver completes a lap, and the lap times determine the race start position. This shows the qualifying position and final race finishing position of drivers for every race going back to 1996 and has 7625 datapoints. The data was once again accessed using the Ergast F1 API using this python script. A heatmap was used because of the number of datapoints and recurring values. This shows a clear positive correlation between qualifying position and final race outcome (showed by a higher number of records on the diagonal).

Conclusion: What does this tell us?


At the start of this project, I aimed to analyse the possible correlation between R&D expenditure and constructor standings in Formula 1. This relationship was clearly shown and - unsurprisingly - the richer the team, the better they tend to perform. This is the reason why teams such as Mercedes, Red Bull and Ferrari with huge budgets are in a different league to the rest of the teams in races. In 2022, a budget cap is going to be set which will limit the amount that teams can spend on R&D, to level the playing field. This project has shown that such as budget cap will likely have a significant impact on the inequality in constructor performance. Further work can be done here to add more variables and build a stronger model to understand what drives constructors’ performance in Formula 1, such as driver skill (which is difficult to measure), history of the team (whether they have a rich history of car manufacturing, such as Mercedes, or not) and more.
